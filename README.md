# AIWorkoutPLanner

This project is a frontend-only AI workout generator built with basic HTML, CSS, and JavaScript. It creates a personalized workout plan based on duration, goal, difficulty, and mixed workout preferences.

## Files

- `index.html` — the app structure and form inputs for duration, goal, difficulty, and mixed mode.
- `styles.css` — the visual layout and table styling for a clean workout plan display.
- `script.js` — the workout generation logic and table rendering.

## Code overview

- `script.js` contains:
  - data pools for warm-up, main exercise, and cooldown options,
  - difficulty scaling rules for sets/reps and cardio work/rest intervals,
  - functions to build strength, cardio, and mixed plans,
  - DOM rendering logic to update the workout table when the form is submitted.

## Running the app

Open `index.html` directly in your browser. No backend or server is required.
